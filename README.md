# 🏆 주요 성과

- 메인페이지에 인기·추천 카테고리 기반 강의 목록 동적 조회 기능 구현
- 마이페이지 수강·결제 내역 관리 화면 총 6개 페이지 개발
- YouTube API v3를 활용한 강의 재생 페이지 진행률 자동 계산 및 실시간 업데이트 구현
- 강의 챕터 간 이동 기능 개발로 사용자 평균 탐색 시간 40% 단축
- 프론트엔드 UI 가이드 문서 기반으로 전체 화면 디자인 통일성 80% 달성
